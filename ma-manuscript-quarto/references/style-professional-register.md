# Rule 2: Professional Academic Register

**Parent**: [Academic Writing Style Guide](academic-writing-style.md)

---

## 2.1 Complete Sentence Structure

Every sentence in the manuscript must be grammatically complete with subject, verb, and (where appropriate) object.

**Pattern for reporting results** (use consistently):

```
[Subject] [verb in past tense] [object] ([effect estimate], 95% CI [lower]-[upper]; [heterogeneity]).
```

Examples:

- "The pooled risk ratio favored the intervention group (RR 1.26, 95% CI 1.16-1.37; I^2^ = 0%, p = 0.0015)."
- "Subgroup analysis by PD-L1 status revealed no significant interaction (p~interaction~ = 0.34)."
- "Sensitivity analysis excluding the open-label trial did not materially alter the primary estimate (RR 1.24, 95% CI 1.13-1.36)."

## 2.2 Appropriate Use of Passive Voice

Scientific manuscripts use passive voice when the agent is irrelevant or when convention dictates. Active voice is preferred when it improves clarity.

**Use passive voice for**:

| Context | Example |
|---------|---------|
| Methods (convention) | "Studies were identified through a systematic search of PubMed, Embase, and the Cochrane Library." |
| Results (focus on data) | "A statistically significant improvement was observed in the intervention group." |
| Procedures | "Risk of bias was assessed using the Cochrane RoB 2 tool." |
| Standard reporting | "Heterogeneity was quantified using the I^2^ statistic." |

**Use active voice for**:

| Context | Example |
|---------|---------|
| Author actions | "We included five randomized controlled trials enrolling 2,402 patients." |
| Interpretation | "These findings suggest that the addition of immunotherapy confers a meaningful survival benefit." |
| Comparison with others | "Our results extend the findings of Smith et al. by including three additional trials published after their review." |

**Guideline**: Default to passive in Methods and Results; use active in Introduction (framing the gap) and Discussion (interpretation).

## 2.3 Formal Transitions

Replace casual connectors with formal academic transitions. Do not overuse any single transition.

| Casual / Overused | Formal Alternatives |
|-------------------|---------------------|
| Also | In addition, Furthermore, Moreover |
| But | However, Nevertheless, Conversely, In contrast |
| So | Therefore, Consequently, Accordingly, Thus |
| Plus | Additionally, In addition |
| On top of that | Furthermore, Moreover |
| In the end | Ultimately, In summary |
| On the other hand | Conversely, By contrast, Alternatively |
| Because of this | As a consequence, As a result, For this reason |
| Even so | Nonetheless, Nevertheless, Despite this |
| What's more | Furthermore, Additionally |

**Transition placement rules**:

1. Place transitions at the beginning of a sentence, followed by a comma.
2. Vary transitions within a section; do not use "However" more than twice per subsection.
3. Do not begin more than two consecutive sentences with a transition word.
4. Prefer sentence-level logical connectors ("These results are consistent with...") over standalone transition adverbs when possible.

## 2.4 Precise Scientific Language

Replace general terms with domain-specific vocabulary.

| General / Imprecise | Precise Scientific |
|---------------------|-------------------|
| showed | demonstrated, revealed, indicated |
| better | superior, more favorable, improved |
| worse | inferior, less favorable, poorer |
| a lot / many | numerous (qualitative), N (quantitative) |
| patients | participants, subjects, individuals (per journal style) |
| drugs | therapeutic agents, pharmacologic interventions |
| treatment worked | the intervention was efficacious / effective |
| side effects | adverse events, adverse effects, toxicities |
| death | mortality, all-cause mortality, disease-specific mortality |
| came back | recurred, relapsed |
| spread | metastasized, disseminated |
| test | assay, assessment, evaluation, analysis |
| check | verify, assess, evaluate, examine |
| link | association, correlation, relationship |
| cause | etiology, pathogenesis (for disease); confer, contribute to (for risk) |

**Numbers and statistics**:

- Always report exact values: "12 studies" not "several studies"; "p = 0.03" not "p < 0.05" (unless convention requires the latter).
- Always include units: "median follow-up of 36 months" not "median follow-up of 36".
- Report confidence intervals with the estimate: "HR 0.78 (95% CI 0.65-0.93)" not "HR 0.78, which was significant".
- Do not use "significant" without specifying statistical or clinical significance.
