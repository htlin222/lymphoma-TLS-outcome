# Rule 1: No AI Casual Writing

**Parent**: [Academic Writing Style Guide](academic-writing-style.md)

Medical manuscripts must read as if written by a domain expert, not generated by an AI assistant. The following patterns are **prohibited** in all manuscript sections.

---

## 1.1 No Contractions

Contractions signal informal register and are unacceptable in peer-reviewed manuscripts.

| Prohibited | Replacement |
|------------|-------------|
| don't | do not |
| didn't | did not |
| doesn't | does not |
| wasn't | was not |
| weren't | were not |
| can't | cannot |
| won't | will not |
| wouldn't | would not |
| shouldn't | should not |
| couldn't | could not |
| it's (it is) | it is |
| that's | that is |
| there's | there is |
| we've | we have |
| they've | they have |
| we're | we are |
| they're | they are |
| who's | who is |
| let's | let us |

**Validation**: Search all `.qmd` files for `n't`, `'s`, `'re`, `'ve`, `'ll` patterns. Must return 0 hits (excluding possessives and proper nouns such as Crohn's, Hodgkin's).

---

## 1.2 No Vague or Hedging Language

Every claim must be precise. Vague language weakens scientific arguments and invites reviewer criticism.

| Prohibited | Why | Replacement |
|------------|-----|-------------|
| figure out | colloquial | determine, elucidate, ascertain |
| maybe | non-committal | possibly, potentially, it is plausible that |
| a lot of | imprecise | numerous, a substantial proportion of, N studies |
| kind of / sort of | hedging | somewhat, partially, to a degree |
| pretty much | colloquial | largely, substantially, approximately |
| basically | filler | (delete, or rewrite the sentence) |
| stuff / things | imprecise | factors, variables, components, outcomes |
| big / small | imprecise | substantial, considerable / modest, minimal |
| get / got | informal | obtain, acquire, achieve, receive |
| find out | informal | determine, identify, ascertain |
| look at | informal | examine, evaluate, assess, investigate |
| a bit | informal | slightly, marginally, modestly |
| seems like | hedging | appears to, suggests that |
| try to | weak | aim to, seek to, attempt to |

**Rule**: If a sentence contains a vague term, rewrite the entire sentence rather than performing a word-level swap.

---

## 1.3 No Command Voice (Imperative Mood)

Manuscripts describe what was done and what was found. They do not issue instructions.

| Prohibited | Why | Replacement |
|------------|-----|-------------|
| "Run the analysis." | imperative | "The analysis was performed using..." |
| "Don't count these." | imperative + contraction | "These were excluded from the count." |
| "Consider the bias." | imperative | "The potential for bias warrants consideration." |
| "Note that..." | imperative | "It should be noted that..." / "Notably,..." |
| "See Figure 1." | imperative | "As shown in Figure 1..." / "Figure 1 illustrates..." |
| "Compare Table 2." | imperative | "A comparison is presented in Table 2." |
| "Assume normality." | imperative | "Normality was assumed based on..." |

**Exception**: The phrase "see" is acceptable in parenthetical cross-references: "(see Supplementary Table S1)" is standard journal convention.

---

## 1.4 No Sentence Fragments

Every sentence must contain a subject and a predicate. Fragments are acceptable in bullet lists and tables but never in manuscript prose.

| Prohibited | Why | Replacement |
|------------|-----|-------------|
| "Next steps:" | fragment | "The following steps were undertaken." |
| "No data." | fragment | "No data were available for this outcome." |
| "Similar results." | fragment | "Similar results were observed across subgroups." |
| "Not significant." | fragment | "The difference did not reach statistical significance." |
| "As expected." | fragment | "As expected, the pooled estimate favored the intervention." |
| "More studies needed." | fragment | "Additional studies are required to confirm these findings." |
