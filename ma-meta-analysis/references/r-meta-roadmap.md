# r-meta Roadmap Summary

Core tasks for the analysis workflow:
- Read and clean the extraction dataset
- Compute effect sizes
- Fit fixed and random effects models
- Produce forest plots
- Assess heterogeneity (I2, tau2, Q)
- Run subgroup and sensitivity analyses
- Run meta-regression when appropriate
- Evaluate publication bias with funnel plots and tests
- Produce manuscript-ready tables and figures
